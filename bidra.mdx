Vi arbeider kontinuerlig med å dokumentere cybersikkerhetshendelser i Norge på vår nye nettside. Hvis du har informasjon om en hendelse som vi mangler, eller om du har ytterligere detaljer om hendelser vi allerede har nevnt, vil vi sette stor pris på om du deler den med oss!

For å gjøre nettsiden vår så nyttig som mulig, ønsker vi følgende informasjon om hver hendelse:

- Hvem ble utsatt for cybersikkerhetshendelsen?
- Når skjedde det?
- Hva slags hendelse var det? (f.eks. DDoS angrep, løsepengevirus, stjålet påloggingsinformasjon osv.)
- Hvem står bak hendelsen?
- Lenker! Vi setter stor pris på lenker til nyhetsartikler, pressemeldinger og lignende kilder.

Send inn info på e-post til `datainnbrudd[at]proton[dot].me` eller opprett en issue/PR gjennom vårt GitHub repo. Vi viser kun offentlig kjente hendelser og informasjon som er klassifisert som [TLP:WHITE](https://www.cisa.gov/news-events/news/traffic-light-protocol-tlp-definitions-and-usage). Vær derfor forsiktig med å ikke avsløre informasjon som ikke er TLP:WHITE.

Tusen takk for din hjelp! ❤️
